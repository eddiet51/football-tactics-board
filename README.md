# ⚽ 足球戰術版 Football Tactics Board

互動式足球戰術板，支援多種陣型和戰術線繪製。

## 🎯 功能特色

- ✅ **新增**：可切換**直式 (Portrait)** 或 **橫式 (Landscape)** 球場佈局
- ✅ 可拖曳球員位置
- ✅ 6 種常用陣型選擇（4-3-3, 4-4-2, 3-5-2, 5-3-2, 4-2-3-1, 3-4-3）
- ✅ 繪製戰術線條
- ✅ 可移動足球圖示
- ✅ 替補球員固定顯示
- ✅ 支援觸控螢幕操作
- ✅ 完整的足球場標記（中線、禁區、罰球弧、球門等）

## 🚀 線上使用

訪問：`https://YOUR_USERNAME.github.io/football-tactics-board/`

直接開啟 `index.html` 即可使用，無需安裝任何套件。

## 🎮 操作說明

### 佈局與陣型選擇
- **球場方向**：使用「球場方向」選單切換直式或橫式，預設為直式。
- **陣型選擇**：分別選擇紅隊與藍隊的陣型。
- 選擇方向或陣型後，球員位置和代號會自動更新。

### 球員操作
- **拖曳球員**：點擊並拖曳場上球員到任意位置
- **移動足球**：拖曳足球圖示到任意位置

### 戰術線繪製
- **開始畫線**：右鍵點擊第一個球員
- **完成畫線**：右鍵點擊第二個球員
- **取消畫線**：按 ESC 鍵

### 控制按鈕
- **重置位置**：將所有球員恢復到初始位置
- **清除線條**：清除所有戰術線
- **切換隊伍**：交換紅藍兩隊位置

### 鍵盤快捷鍵
- `R` - 重置位置
- `C` - 清除線條
- `ESC` - 取消畫線

## 📋 可用陣型

| 陣型 | 說明 |
|------|------|
| **4-3-3** | 經典進攻陣型，3 名前鋒 |
| **4-4-2** | 平衡陣型，2 名前鋒 |
| **3-5-2** | 中場控制陣型，5 名中場 |
| **5-3-2** | 防守陣型，5 名後衛 |
| **4-2-3-1** | 現代陣型，雙後腰 + 攻擊中場 |
| **3-4-3** | 進攻陣型，3 名後衛 + 3 名前鋒 |

## 🛠️ 技術架構

- **HTML5** - 頁面結構
- **CSS3** - 樣式設計（漸層、陰影、動畫）
- **Vanilla JavaScript** - 互動功能（無外部依賴）
- **Canvas API** - 戰術線繪製

- 戰術線會隨球員移動自動更新
